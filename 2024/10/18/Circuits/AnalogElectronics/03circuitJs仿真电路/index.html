欧姆定律

  欧姆定律表明，电流（I）通过导体的强度与施加在其两端的电压（V）成正比，与导体的电阻（R）成反比。公式为：𝑉=𝐼×𝑅

电阻

  串联和并联电阻的计算方法如下：

  • 串联电路 ： 在串联电路中，总电阻 R总R_总R总​ 就是各个电阻之和：
    • R总=R1+R2+R3+RnR_总 = R_1 + R_2 + R_3 + R_nR总​=R1​+R2​+R3​+Rn​
  • 并联电阻 : 在并联电路中，总电阻 R总R_总R总​ 的倒数等于各个电阻倒数之和：
    • 1R总=1R1+1R2+1R3+1Rn\frac{1}{R_总} = \frac{1}{R_1} + \frac{1}{R_2} + \frac{1}{R_3} + \frac{1}{R_n}R总​1​=R1​1​+R2​1​+R3​1​+Rn​1​，可以通过取倒数来得到总电阻：R总=11R1+1R2+1R3+1RnR_总 = \frac{1}{\frac{1}{R_1} + \frac{1}{R_2} + \frac{1}{R_3} + \frac{1}{R_n}}R总​=R1​1​+R2​1​+R3​1​+Rn​1​1​

电容

  电容是储存电能的元件，由两个导体之间的绝缘材料（介质）构成。其主要作用包括：

  1. 储能 ：电容可以储存电能并在需要时释放，常用于平滑电源波动。
  2. 滤波 ：在电源电路中，电容器可以滤除高频噪声，保持稳定电压。
  3. 耦合与解耦 ：用于信号的耦合与解耦，允许交流信号通过而阻止直流成分。
  4. 定时 ：在某些电路中与电阻一起使用，形成定时电路。

电容的主要作用是防止电压突变

电感器

  电感器是一种储存磁能的元件，由导线绕成线圈构成。当电流通过时，电感器会在其周围产生磁场。其主要作用包括：

  1. 储能 ：电感器可以储存能量，当电流变化时，它会释放能量，平滑电流波动。
  2. 滤波 ：在电源电路中，电感器可以抑制高频信号，帮助实现低通滤波。
  3. 阻抗匹配 ：在射频和音频电路中，电感器用于阻抗匹配，提高信号传输效率。
  4. 耦合与解耦 ：与电容器一起使用，电感器可以实现信号的耦合与解耦。

电感的主要作用是防止电流突变

LRC振荡电路

  LRC电路是由电感（L）、电容（C）和电阻（R）组成的电路。它的特性和行为取决于这些元件的组合，通常用于研究交流电路的频率响应和振荡行为。

  LRC电路的主要特性：

  1. 谐振 ：在特定频率（谐振频率）下，电感和电容的反应相互抵消，使电路呈现最小阻抗。
    • 谐振频率计算公式为：f0=12πLCf_0 = \frac{1}{2\pi\sqrt{LC}}f0​=2πLC​1​
  2. 阻抗 : LRC电路的总阻抗Z取决于频率f、电阻R、电感L和电容C
    • 阻抗计算公式为：Z=R2+(XL−XC)2Z = \sqrt{R^2 + (X_L - X_C)^2}Z=R2+(XL​−XC​)2​，其中XL=2πfLX_L = 2 \pi fLXL​=2πfL是电感的反应，XC=12πfCX_C = \frac{1}{2 \pi f C}XC​=2πfC1​是电容的反应
  3. 相位差 ： 电流和电压之间的相位差也受到电感和电容的影响。在谐振频率下，相位差为零。

  电阻分压，观察不同位置的电压

可变电阻

  电阻可以拖动改变大小

电位器分压器

戴维宁定理

  任何复杂电路可以等效为一个电压源和一个电阻器组成

诺顿定理

  任何复杂电路可以等效为一个电流源和一个电阻器组成

基尔霍夫定律

  对于电路中任一节点流入节点的电流之和必等于流出该节点的电流之和
